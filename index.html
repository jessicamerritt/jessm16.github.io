My Co-op

I completed my first Co-op at Novartis Institutes of BioMedical Research where I implemented full stack solutions using JavaScript (React.js, Redux, and Backbone.js), Java, and SQL. Working at NIBR allowed me to collaborate with an international team in order to deliver informatics solutions to scientists. I worked primarily on an online lab information management system which enabled collaboration between labs and captured data for future machine learning. The application's front end was written in Backbone, but the team was considering migrating it to React. I completed a proof of concept by integrating React.js code into the Backbone code base through JQuery plugins to allow for a temporary fix. I also developed and designed a lambda function using the serverless computing stack on AWS which combined an existing set of endpoints into a single study export. My careful design enabled users to export more information than the existing study export while also halving the size of the export.
